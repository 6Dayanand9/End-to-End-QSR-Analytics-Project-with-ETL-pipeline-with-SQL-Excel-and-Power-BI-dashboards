# End-to-End-QSR-Analytics-Project-with-ETL-pipeline-with-SQL-Excel-and-Power-BI-dashboards


## 🚀 Project Overview

Organizations often struggle to understand whether business growth is being driven by sustainable profitability or masked by excessive discounts, trade spend, and operational inefficiencies.

This project delivers an end-to-end **Budget vs Actual Financial Analytics Platform** that transforms raw financial data into actionable business insights across revenue, profitability, costs, channels, products, and locations.

The solution analyzes:

* **500,000+ records**
* **5 years of historical data**
* **4,000+ SKUs**
* Multiple channels, locations, and customer segments

Built using:

`Power BI | MySQL | Excel| DAX | Power BI Service`
